## Getting Started

### Prerequisites

- Node.js (v16+)
- PNPM package manager
- PostgreSQL

### Installation

```bash
# Install all dependencies
pnpm install
````

### Development

```bash
# Run both frontend and backend in development mode
pnpm dev

# Or run them separately
pnpm frontend:dev
pnpm backend:dev
```

## Tech Stack

- **Frontend**: Next.js, TypeScript, React
- **Backend**: Express.js, TypeScript, Prisma ORM
- **Database**: PostgreSQL
